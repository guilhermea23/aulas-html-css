# HTML ✔
  
  - Hyper-Text-Markup-Language ou Linguagem de Marcação de HiperTexto
  - O corpo de uma página.

# Tags HTML ✔
  - Como se trata de uma linguagem de marcação, nós apenas delimitamos onde algum elemento deve ficar.
  - Podemos delimitar:
    - Títulos 
    - Barra de Navegação
    - Parágrafos (Avisos!)
    - Imagens
    - Links
    - Caixas de Texto
    - ...
